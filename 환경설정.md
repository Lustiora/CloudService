## 클라우드 실습 프로젝트

### 프로젝트의 기본 디렉토리
1) bin
  - www 파일 존재. 서버를 시작할 포트가 지정됨.
2) node_modules
  - npm i > 실행시 생성. package.json에 선언된 모듈과 이 모듈을 실행하기 위해 필요한 의존관계 모듈이 설치됨.
3) public
  - image, JavaScript, StyleSheet 같은 정적 리소스 관리.
4) routes
  - index.js, users,js
  - 기본 생성 파일. 라우트를 처리하기 위한 코드 있음.
5) views
  - views를 처리하는 파일 위치.
  - Project 생성시 Template 엔진으로 ejs를 지정.
  - 이 디렉토리에는 확장자가 ejs인 파일이 위치.
